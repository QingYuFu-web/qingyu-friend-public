# 响应速度优化方案 🐢 -> 🐇

树莓派 5 运行 1.5B 模型确实有几秒延迟。如果觉得慢，我们有两个选择：

## 方案 A：极致速度（切换回 0.5B）⚡
- **优点**：秒回，像聊天软件一样快。
- **缺点**：在这之前你体验过，有时候会答非所问（"笨"）。
- **适合**：日常简单打招呼、开关灯指令等。

## 方案 B：云端大脑（DeepSeek/OpenAI API）☁️
- **优点**：又快又聪明（比 1.5B 聪明 100 倍）。
- **缺点**：需要花钱（DeepSeek 很便宜，几块钱能用很久），必须联网。
- **适合**：深度聊天、复杂任务。

## 推荐操作

既然你觉得 1.5B 慢，我们先**切回 0.5B** 试试现在的"完整代码"版本。之前的"笨"可能是代码没写好导致的，现在我们有了完善的 Prompt 和记忆系统，0.5B 的表现可能会好一些。
